# Professional Portfolio Starter

This is a clean, professional portfolio starter designed for software engineering job applications.

## Files
- `index.html` - Main portfolio page
- `styles.css` - Site styling
- `script.js` - Small script for dynamic footer year
- `resume-template.md` - Resume content template with impact-focused bullets

## Quick start
Open `index.html` in your browser.

## Personalization checklist
1. Replace placeholder text in `index.html`.
2. Add real project links and metrics.
3. Update contact links.
4. Rename `resume-template.md` to your final resume content file and export to PDF.
5. Buy a custom domain and deploy (GitHub Pages / Vercel / Netlify).
